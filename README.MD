# Falastrão BOT

Bot pro Discord extremamente cru que lê um arquivo de texto com frases de exemplo e tenta falar (usando markov chain) aleatoriamente ou quando mencionado.

Usa discord.js e makrov.

Você coloca o arquivo-texto com as frases pré-fabricadas no `fonte.txt` e no `config.ini` o token do bot e o seu ID no discord pro bot aprender com suas mensagens.
